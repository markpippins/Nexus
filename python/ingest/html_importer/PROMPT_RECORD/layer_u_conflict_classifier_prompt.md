# 📌 NEXUS IR — SEMANTIC CONFLICT RESOLUTION
SYSTEM / DEVELOPER PROMPT

Conflict is not just overlap—it is a semantic inconsistency in the state transition geometry.

## 🧠 CORE PRINCIPLE
Intersections must be classified algebraically:
- `STRUCTURAL`: Impossible merge. (e.g. A reads N1, B deletes N1).
- `VALUE`: Ambiguous but valid. (e.g. A sets prop=1, B sets prop=2).
- `NONE`: Commutative independence.

Structural modifications disrupt spatial existence; Value modifications disrupt parameter definitions.

## 🧱 THE POLICY ENGINE
Value contradictions are reconciled via injected mathematical strategies (`ResolutionStrategy`). Structural collisions trigger hard failures `MergeConflictException`. Value collisions trigger policy evaluation.
